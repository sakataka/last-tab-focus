# Chrome Web Store Descriptions

## en

Last Tab Focus is a small Chrome extension built around one specific behavior.

When you close the current tab, it moves focus back to the tab you were using just before it.

That is all it does.

It is not a general tab manager. It does not add side panels, tab groups, sessions, saved workspaces, sync features, or other unrelated tools.

The goal is to keep tab closing predictable without bringing in extra behavior or unnecessary surface area.

It stays local:

- No analytics
- No external communication
- No remote services
- No browsing data sent anywhere

## zh_CN

所有行为都保留在本地。

- 无外部通信
