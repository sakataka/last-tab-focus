# Chrome Web Store Descriptions

## en

Last Tab Focus is a small Chrome extension for one specific tab behavior.

When you close the current tab, Chrome returns to the previously active tab in the same window.

If you often close tabs with Ctrl+W or Command+W and expect to go back to the last tab you were using, this extension keeps that workflow predictable.

That is all it does.

It is not a general tab manager. It does not add side panels, tab groups, session management, saved workspaces, sync features, or other unrelated tools.

The goal is to make tab closing feel natural without bringing in extra behavior or unnecessary surface area.

Main behavior:

- Return to the previous tab after closing the current tab
- Track the last active tab within the same Chrome window
- Keep tab focus predictable when closing several tabs in a row
- Stay small and single-purpose

It stays local:

- No analytics
- No external communication
- No remote services
- No browsing data sent anywhere

## zh_CN

Last Tab Focus 是一个很小的 Chrome 扩展，只专注于关闭标签页后的一个行为。

当你关闭当前标签页时，它会在同一个窗口中回到刚刚使用过的上一个标签页。

如果你经常用 Ctrl+W 或 Command+W 关闭标签页，并希望回到最近使用的标签页，而不是左边或右边的标签页，这个扩展可以让标签页焦点更符合习惯。

它只做这一件事。

它不是完整的标签页管理器。它不会添加侧边栏、标签页分组、会话管理、工作区保存、同步功能或其他无关工具。

它的目标是让关闭标签页后的返回位置更自然，同时尽量减少额外行为和不必要的副作用。

主要行为：

- 关闭当前标签页后回到上一个标签页
- 在同一个 Chrome 窗口中跟踪最近使用的标签页
- 连续关闭多个标签页时，让标签页焦点更容易预测
- 保持轻量，只做一个功能

所有行为都保留在本地。

- 无分析统计
- 无外部通信
- 无远程服务
- 不会向任何地方发送浏览数据
