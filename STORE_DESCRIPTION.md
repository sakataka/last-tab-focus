# Chrome Web Store Description

Last Tab Focus does one thing.

When you close the current tab, it moves focus back to the tab you were using just before it.

It is not a general tab manager. It does not add side panels, tab groups, sessions, or other unrelated features.

It only changes how tab closing behaves in the same window.

It also stays local:

- No analytics
- No external communication
- No remote services
- No browsing data sent anywhere
